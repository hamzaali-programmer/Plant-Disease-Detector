# 🌾 Wheat Plant Disease Detector

A Machine Learning based web application that detects wheat leaf diseases using image classification.

## Features

- Upload wheat leaf image
- Detect disease
- Confidence score
- Treatment advice
- Beautiful Streamlit UI

---

## Diseases

- 🟢 Healthy
- 🟤 Brown Rust
- 🟡 Yellow Rust

---

## Technologies

- Python
- Streamlit
- TensorFlow
- MobileNetV2
- Logistic Regression
- OpenCV
- Scikit-learn

---

## Project Structure

```
Plant_Detector/
│
├── app.py
├── train_model.py
├── plant_model.pkl

├── requirements.txt
├── README.md
```

## Run Project

```bash
pip install -r requirements.txt
streamlit run app.py
```
